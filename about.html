关于我​

你好！我是星矢，英文名Xingshi，名字源于自己的真实姓名 xingshuai 二字的谐音，目前是一位基础的前端开发工程师，平时从事 Shopify 相关的开发，一个正在努力学习 React 相关技术栈的“程序猿”

没有什么拿得出手的兴趣爱好，不打游戏，平时喜欢做一些自己喜欢的事情，休息日出去溜达、捣鼓自己的网站、听音乐等等，算是一个有点小无趣的人吧，不过自己特喜欢一些突发奇想 然后去做的事情

比如这个网站的由来，就是自己想要一个独属于自己的空间，去记录自己的成长、总结学到技术的文档以及分享自己的一些碎碎念的地方

关于本站​

本站是一个基于 Docusaurus 构建 托管在 Github Page 上面 用 Cloudflare的免费DNS服务解析域名的静态网站

目前技术尚且浅薄，用的就是 Docusaurus默认的样式去构建的，包括此时这个page也是用 Markdown 文件写的

选用 Docusaurus构建网站除了因为它是基于 React 开发的 可以借此机会去学习 React外，能用简单的 .md文件去构建一些页面、写一些文章，也算是一些原因吧，后续技术进步一些，会尝试对网站进行二开

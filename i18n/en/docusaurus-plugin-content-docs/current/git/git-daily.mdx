# Git daily summary

When using `git` in the IDE, you may encounter a situation where the online warehouse has but the local warehouse does not have a certain branch. In this case, you need to execute the `git fetch` command or the `fetch` on the GUI interface to update the code.
![git-daily01.webp](https://github.com/xingshi012/blog-images/blob/main/docs/git-daily/git-daily01.webp?raw=true)

When executing `revert` in IDEA, you may encounter a local rollback of the code. However, because it is associated with the online code, the code cannot be synchronized online, so you need to execute `git push -f` to force the code to be pushed online.
![git-daily02.webp](https://github.com/xingshi012/blog-images/blob/main/docs/git-daily/git-daily02.webp?raw=true)

`git` is a tool born in Linux system, so there are some commonalities. For example, some commands may report errors when run in the terminal of Windows system, but they run normally in `git bash`.
![git-daily03.webp](https://github.com/xingshi012/blog-images/blob/main/docs/git-daily/git-daily03.webp?raw=true)

## Some routes that come with Docusaurus

At the moment (2026-01-25) the technology is still shallow, and the blog is built using the default theme of `Docusaurus` and has not yet undergone secondary development\
So here are some useful routes that come with Docusaurus, recorded to prevent them from being found later:

- `/blog`  blog template default blog directory

- `/blog/archive` Blog template default archive directory

- `/blog/tags` blog template default tag directory
  
- `/docs` docs template default docs directory

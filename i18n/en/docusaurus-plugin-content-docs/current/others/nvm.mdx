# NVM Common Commands

> Summary of commonly used nvm commands


`nvm` is a version management tool for `node`, allowing you to conveniently install and switch between different Node.js versions when working in environments that require different Node versions.

## Installation

You can download nvm from the official website:\
https://www.nvmnode.com/guide/download.html

After installation, run `nvm -v` to verify that `nvm` is installed correctly.

![nvm-01.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-01.webp?raw=true)

## Node Version Management

Use `nvm ls` to view all locally installed Node versions.

![nvm-02.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-02.webp?raw=true)

Use `nvm list available` to view all available Node versions that can be installed.

![nvm-03.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-03.webp?raw=true)

## Node Version Switching

Use `nvm use <version>` to switch to a specific Node version.

![nvm-04.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-04.webp?raw=true)

## Uninstalling Node

Use `nvm uninstall <version>` to uninstall a specific Node version.

![nvm-05.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-05.webp?raw=true)

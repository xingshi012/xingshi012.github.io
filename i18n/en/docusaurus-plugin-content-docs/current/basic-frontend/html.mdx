HTML is the most basic content of front-end development. The final display effect of all web pages is the DOM structure of HTML.

HTML tags are divided into `inline elements`, `block-level elements` and `inline block elements`, which are divided into `single tags` and `double tags`.

In daily development, commonly used tags include `div tag` `span tag` `p tag` `a tag` `h series tag` `img tag` `video tag` `input tag` and so on.

It is best to focus on semantics in HTML development. For example, on the homepage of a website: the top header can be used as a navigation with the `header tag`, and the nav tag is used inside, and the main content is wrapped in the `main tag`.
Then in the `main tag`, each section can be wrapped by a `section tag`, and the largest content in each `section tag` can use the `h tag`, and then the content can use the `p tag`, and the button can use the `button tag`.
This level is probably written like this. The semantics are relatively clear and easy for SEO, accessibility optimization, etc.

```
<body>
  <header>
    <nav>
      <h1>
        <a href="#">
          <img src="logo.png" alt="Logo" title="Logo" />
        </a>
      </h1>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <section>
      <h2>Heading</h2>
      <div>
        <p>
          This is some description.
        </p>
        <button>Get Started</button>
      </div>
    </section>
  </main>

  <footer>
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
    <p>Copyright </p>
  </footer>
</body>
```

There will be some attributes on HTML tags. Basic attributes such as `class` `id` attributes can be selected through `css selector`, and then some styles can be applied. According to different HTML tags, there will be some other attributes, such as `src` `alt` `title` attributes of `img` tag, etc.
Some special attributes, such as `aria-label` and `tabindex`, are used for accessibility. Some custom attributes starting with data- can be combined with `js` to implement some small functions.

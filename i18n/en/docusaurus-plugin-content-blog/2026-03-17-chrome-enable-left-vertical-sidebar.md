> This function is currently (2026-03-17) still in the experimental stage. It is not as stable as the default mode and maybe not perfect yet. 

<!-- truncate -->

Update chrome browser to latest version

![chrome-enable-left-vertical-sidebar01-01.webp](https://github.com/xingshi012/blog-images/blob/main/blogs/2026/2026-03-17/chrome-enable-left-vertical-sidebar01-01.webp?raw=true)

Type in the address bar `chrome://flags` 

![chrome-enable-left-vertical-sidebar02.webp](https://github.com/xingshi012/blog-images/blob/main/blogs/2026/2026-03-17/chrome-enable-left-vertical-sidebar02.webp?raw=true)

Search for `Vertical Tabs` to enable it and restart the browser

![chrome-enable-left-vertical-sidebar03.webp](https://github.com/xingshi012/blog-images/blob/main/blogs/2026/2026-03-17/chrome-enable-left-vertical-sidebar03.webp?raw=true)

**Right click on the blank space of the top tab bar:**

Click the `move tabs to the side`

![chrome-enable-left-vertical-sidebar04-04.webp](https://github.com/xingshi012/blog-images/blob/main/blogs/2026/2026-03-17/chrome-enable-left-vertical-sidebar04-04.webp?raw=true)
Or in this address `chrome://settings/appearance` 

Set the `Tab strip position` as the `Side`

![chrome-enable-left-vertical-sidebar05-05.webp](https://github.com/xingshi012/blog-images/blob/main/blogs/2026/2026-03-17/chrome-enable-left-vertical-sidebar05-05.webp?raw=true)

That's it!

![chrome-enable-left-vertical-sidebar06.webp](https://github.com/xingshi012/blog-images/blob/main/blogs/2026/2026-03-17/chrome-enable-left-vertical-sidebar06.webp?raw=true)

**It is still in test mode and the function may not be perfect**

# Markdown 相关

> `Markdown` 语法相关

## 一些 markdown 语法

换行
> 一段文字的结尾处用一个这个可以换行 `\`\
> 类似html的 `<br>` 标签


*斜体*
> 文字前后各一个星号包裹文字，以实现斜体效果\
> `*斜体*`

**粗体**
> 文字前后各两个个星号包裹文字，以实现斜体效果\
> `**粗体**`

list:
列表使用 `-` 开头，以实现列表效果\
> - 内容1
> - 内容2
> - 内容3

列表使用 `1.` 开头，以实现有序列表效果\
> 1. 内容1
> 2. 内容2
> 3. 内容3

引用\
列表使用 `>`和一个空格 开头，以实现引用效果\
> `> 引用内容`

链接\
列表使用 `[]()` 包裹内容，以实现链接效果\
> `[链接](链接地址)`\
[点我跳转首页](/)

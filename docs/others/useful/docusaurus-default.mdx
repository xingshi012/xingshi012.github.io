## Docusaurus自带的一些路由

当下（2026-01-25）技术尚且浅薄，用的是 `Docusaurus` 默认主题去搭建的博客 尚未进行二次开发\
所以这里是一些Docusaurus自带好用的几个路由，记录下来 防止后续找不到的：

- `/blog`  blog模板默认博客目录

- `/blog/archive` blog模板默认归档目录

- `/blog/tags` blog模板默认标签目录
  
- `/docs` docs模板默认文档目录

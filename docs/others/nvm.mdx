# nvm 常用命令

> nvm 常用命令总结


`nvm` 是 `node` 的版本管理工具，可以便捷的安装、切换不同的 node 版本，在需要不同的 node 版本的环境下。

## 安装

可以去 nvm 官网去下载，地址：\
https://www.nvmnode.com/guide/download.html

之后可以执行 `nvm -v` 命令，看下 `nvm` 是否正常安装

![nvm-01.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-01.webp?raw=true)

## node 版本控制

使用 `nvm ls` 命令去查看本地安装的所有 node 版本

![nvm-02.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-02.webp?raw=true)

使用 `nvm list available` 命令查看所有可以安装的 node 版本切换

![nvm-03.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-03.webp?raw=true)

## node 版本切换

使用 `nvm use 版本号` 去切换为具体的 node 版本

![nvm-04.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-04.webp?raw=true)

## node 卸载

使用 `node uninstall 版本号` 去卸载指定的 node 版本

![nvm-05.webp](https://github.com/xingshi012/blog-images/blob/main/docs/nvm/nvm-05.webp?raw=true)

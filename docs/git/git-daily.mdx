# git 日常总结

在 IDE 中使用 `git` 时，会遇到线上仓库有 但是本地仓库没有某一个分支的情况，这时候需要执行 `git fetch`  命令 或者 GUI 界面的 `fetch` 去更新代码
![git-daily01.webp](https://github.com/xingshi012/blog-images/blob/main/docs/git-daily/git-daily01.webp?raw=true)

在 IDEA 中 执行 `revert` 时，可能会遇到本地回退了代码，但是由于和线上关联了，会无法把代码同步到线上，所以需要执行 `git push -f` 强制推送代码到线上
![git-daily02.webp](https://github.com/xingshi012/blog-images/blob/main/docs/git-daily/git-daily02.webp?raw=true)

`git` 属于 Linux 系统诞生的工具，所以会有一些共通性，比如有一些命令在 Windows系统的终端运行可能会报错，但是在 `git bash` 中运行就正常
![git-daily03.webp](https://github.com/xingshi012/blog-images/blob/main/docs/git-daily/git-daily03.webp?raw=true)

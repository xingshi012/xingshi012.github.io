About me​

Hi! I'm Xingshi (星矢), the English name comes from the pronunciation of my Chinese name "xingshuai". I'm currently a frontend developer, mainly working on Shopify related development, and a "programmer" who is trying to learn the React tech stack.

I don't have many impressive hobbies — I don't play games. In my free time, I enjoy doing things I like: going for walks on days off, tinkering with my website, listening to music, etc. I'm kind of a slightly boring person, but I particularly love having spontaneous ideas and then going out to do them.

For example, this website came about because I wanted my own personal space to record my growth, document technical knowledge, and share my scattered thoughts.

About this site​

This site is a static website built with Docusaurus, hosted on GitHub Pages, with domain resolution using Cloudflare's free DNS service.

Currently my technical skills are still quite basic, so I'm using Docusaurus's default styling. Even this page is written with a Markdown file.

I chose Docusaurus not only because it's built on React (giving me an opportunity to learn React), but also because it allows me to create pages and write articles with simple .md files. As my skills improve, I plan to customize the site further.
